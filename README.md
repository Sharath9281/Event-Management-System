# Event-Management-System
The main objective of this document is to illustrate the requirements of the project Event Management system.. The functionality the product must have in order to meet the needs of all stakeholders—businesses and users — is also described. The purpose of this System Requirement Specification (SRS) is to inform readers and users about the features, functions, and specifications of the system. SRS outlines the software's functional, behavioural, and data requirements .The purpose of this software is to oversee the party's activities. This will gather the users' needs for party events. It estimates the total cost of the event based on the user's requirements .Its primary goal is to offer users event-related services at extremely low cost.
